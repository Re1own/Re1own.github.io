Introduction of PQC

PQC(post-quantum cryptography) is to against quantum computer to break cryptography. For instance, RSA can be break by using shor algorithm in quantum computing. Thus, NIST called for some new crypto algorithm which can defense quantum computer breaks such traditional cryptography, they are called post-quantum cryptography.

Evaluation of candidates in the NIST PQC standardization Process

There are many PQC algorithm, so we need a standard to evaluate which PQC is the best and secure enough.

Evaluation Criteria:

  • Keys, Ciphertext, and signstures sizes
  • Security
  • Software/Hardware efficiency
  • Simplicity/flexibility

Five Security level description:

Level1: At least as hard to break as AES-128 using exhaustive key search

Level2: At least as hard to break as SHA-256 using collision search

Level3: At least as hard to break as AES-192 using exhaustive key search

Level4: At least as hard to break as SHA-384 using collision search

Level5: At least as hard to break as AES-256 using exhaustive key search
